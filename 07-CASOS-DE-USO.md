## Limitações e Quando NÃO Usar

### Não Recomendado Para:

1. **E-commerce Completo**
   - Precisa de carrinho, checkout, pagamento
   - CNX é estático, não tem backend
   - **Solução**: Usar CNX para landing pages, integrar com Shopify/WooCommerce para loja

2. **Sites com Muitas Funcionalidades Dinâmicas**
   - Chat em tempo real
   - Sistema de comentários complexo
   - **Solução**: Usar APIs externas (Disqus, etc.)

3. **Sites que Precisam de Backend Complexo**
   - Processamento de dados
   - APIs customizadas complexas
   - **Solução**: Usar CNX para frontend, backend separado

### Recomendado Para:

✅ Blogs e sites de conteúdo  
✅ Sites institucionais  
✅ Portfólios  
✅ Landing pages  
✅ Sites de notícias  
✅ Sites corporativos  
✅ Qualquer site que prioriza performance e simplicidade

**Ideal para:**

**Não ideal para:**
